"Small business owners bidding on government contracts must manually read 50+ page RFP documents and map requirements to their capabilities — an error-prone, time-consuming process where missing one mandatory requirement means automatic rejection."

Designed and built an AI-powered Auto-Proposal Engine using RAG (Retrieval-Augmented Generation) that reads government solicitation documents, extracts atomic requirements, semantically matches them to the user's business profile and past performance, and generates section-by-section compliant proposal drafts — with confidence scores, evidence mapping, and explicit gap flagging to prevent AI hallucination.

  • Full RAG pipeline: PDF parsing → Intelligent chunking → Embeddings → Semantic matching → Controlled LLM generation
  • Hallucination prevention: Evidence-only retrieval + similarity threshold + explicit gap flagging when no match found
  • System prompt engineering: No fabrication rules, compliance priority, requirement-to-evidence mapping enforced
  • Proposal Review UI: Compliance bar, confidence scores, "View Source Clause" links, AI refinement panel — reduces cognitive overload
  • Live MVP: Built and deployed on Lovable with full data schema for SAM.gov API integration
Case Study · 06
AI Face Recognition — PCA + ANN
Computer Vision Python · ML scikit-learn
AI Development Project · End-to-End ML Pipeline
"90,000-dimensional facial image data makes direct classification computationally infeasible."

Built a production-grade face recognition system on a 450-sample dataset by combining PCA dimensionality reduction (eigenfaces) with a multi-layer perceptron ANN — and ran a systematic k-sweep to find the optimal accuracy/efficiency tradeoff.

74%
Peak accuracy at k=100
450
Person dataset built
5
k-values systematically tested
  • Pipeline: Grayscale → Resize → Flatten → PCA → LDA → ANN (Adam optimizer)
  • Key finding: k=100 optimal — more features beyond this introduces noise, not signal
  • PM insight: Feature bloat ≠ better product; ruthless scoping wins — same principle applies to roadmaps
  • Skills: Python, scikit-learn, OpenCV, NumPy, matplotlib, cross-validation
